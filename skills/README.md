# Skills

Custom Claude Code skills for automating workflows.

## Available Skills

### meeting-notes

Process meeting transcripts to create Notion meeting summaries and action items.

**Triggers**: "meeting notes", "process this meeting", "summarize this transcript", or when sharing meeting transcripts.

**Features**:
- Extracts meeting metadata (name, date, category)
- Generates comprehensive, topic-organized notes
- Creates action items with assignees, due dates, and priorities
- Links everything in Notion with detailed context
