# jkramer Plugin

Personal productivity plugin for Claude Code.

## Skills

### meeting-notes

Process meeting transcripts to create Notion meeting summaries and action items.

**Triggers:**
- "meeting notes", "process this meeting", "summarize this transcript"
- Sharing a meeting transcript or recording notes
- Extracting action items from a meeting
- Text that looks like meeting dialogue with team members (Jon, Mark, Toran, Abeer)

**Features:**
- Creates comprehensive, topic-organized meeting notes in Notion
- Extracts action items with assignees, due dates, and priorities
- Links action items to meeting notes via Notion relation
- Adds detailed context to each action item's page content
